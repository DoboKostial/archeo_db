This form creates/defines the evidence of sketches in DB table tab_sketches. The basix evidence item is scanned sketch with references to terrain entities.

There are 3 sections on this page: Section upload. for file upload and referencies definition, Sketches gallery with overview of already uploaded sketches and Stats and filter for quick querying uploaded sketches.

  • Files: (required) is a button for uploading sketches. Enables more files to upload (= they have the same attributes).
  • Sketch type: (required) there are several sketch types: 'sketch', 'photosketch', 'general', 'other'.
  • Author: (required) author of the sketch.
  • Date: Date of a sketch creation.
  • Notes: any kind of additional information.
  • Links: Section to define references to terrain/docu entities

Tips

  • Use -- bulk actions -- to reassign referencies/attributes of more sketches.
