This form creates/updates a polygon by defining ranges of measured point IDs from tab_geopts. The geometry (geom_top/geom_bottom) is rebuilt from those points.

  • Polygon name: unique polygon identifier (primary key).
  • Parent polygon: optional, used for hierarchy.
  • Allocation reason: required, stored in tab_polygons.
  • TOP / BOTTOM ranges: enter one or more FROMâ€“TO intervals (inclusive).

Tips

  • You can define only TOP or only BOTTOM (at least one side is required).
  • Ranges may overlap; points are deduplicated by point ID during rebuild.
  • If geometry is not created, check that points exist in tab_geopts and form a valid polygon.
