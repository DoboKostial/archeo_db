An archaeological object is an interpretive entity composed of multiple stratigraphic units (SU, Fig. 01). One SU can belong to at most one object.

  • ID objekct is unique identifier.
  • Min. 2 SUs are mandatory for defiition/storing of object.
  • Superior object is optional and couldnt create cycle in hierarchy.
  • Object is a grave opens dialogue for defining inhumation grave attributes. (table tab_object_inhum_grave).
