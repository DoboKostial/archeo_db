Upload imports measured points from a TXT/CSV file into tab_geopts, creates bindings (ranges) for the selected side (TOP or BOTTOM) and rebuilds polygon geometry.

Expected file format

id_pts,x,y,h,code,polygon_name
1,-544022.67,-1144012.81,556.1,VP,sonda1
2,-544016.67,-1144022.81,567.1,VP,sonda1
  • polygon_name is taken from the last column (e.g., sonda1).
  • Source EPSG is used to transform XY into the project SRID.
  • Imported side decides whether ranges are saved to TOP or BOTTOM bindings.

Notes

  • Bindings are created automatically from consecutive id_pts blocks in the file.
  • If the same id_pts already exists in tab_geopts, it is updated.
